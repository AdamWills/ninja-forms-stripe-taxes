Adds automatic tax calculation to Ninja Forms Stripe payment forms.

== Description ==

Currently, the Ninja Forms Stripe add on doesn't support automatic tax calculation. This plugin adds that functionality.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the `ninja-forms-stripe-taxes` directory to your `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Profit
